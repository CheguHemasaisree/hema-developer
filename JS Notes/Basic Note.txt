Javascript:- Javascript is a scripting language which is used to build the functionality of the web page
javascript is run on the browser by using an engine called js engine
each and every browser has its own js engine for eg(google chrome has its own engine called v8)
javascript can be run on the client side(Browser) i.e client-side javascript
javascript can be run on the server side(node-js) i.e server-side javascript

how to include javascript into the html
to include the javascript into html we have two ways
1.internal js:-write the code between two tags of script in the bottom of the body
     eg:-   <script>
                console.log("Hello World")
            </script>  
2.external js:-Here the file will be included externally with the filename followed by an extension .js(for eg:sript.js)  
    eg:-script.js(file name)
          console.log("Bye World")
                call the file name into html in this ways
                <script src=./script.js></script>
